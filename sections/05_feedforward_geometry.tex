\section{前馈式统一几何预测与方法趋势}

\subsection{从可微优化到统一前馈推断}

在可微 SfM 之后，VGG 团队研究主线继续向“统一化”与“前馈化”推进。\textit{VGGT: Visual Geometry Grounded Transformer} 提出以单个前馈网络直接预测相机参数、深度图、点图和三维点轨迹等关键几何属性，试图将传统多阶段系统进一步压缩为统一模型\cite{wang2025vggt}。这表明研究问题的表述已经从“如何提升某一几何模块”转向“如何以统一表示组织多个几何任务”。

VGGT 的意义不只在于预测速度更快，更在于它反映出多视图几何重建研究的一个重要趋势：学习模型开始承担传统后端优化原本负责的更大一部分工作。相比于需要测试时迭代求解的系统，前馈式方法在部署、复用和跨任务迁移上具有更高潜力\cite{wang2025vggt}。这也是 VGGT 被视为 VGG 团队研究主线阶段性代表作的重要原因。

从方法论上看，VGGT 代表着一个新的研究假设：在足够强的视觉骨干、足够丰富的训练监督和足够统一的任务设计下，模型能够把原本分散在匹配、相机恢复、三角化与优化中的逻辑，部分吸收到共享表示空间中。这意味着多视图几何重建的核心不再只是“如何设计更复杂的求解器”，而是“如何设计能内化几何规律的统一模型”。因此，VGGT 的价值不仅是速度层面的工程优化，更是问题组织方式的变化。

\subsection{VGGT 的统一输出思想}

与早期将深度估计、位姿估计、轨迹估计分别处理的做法不同，VGGT 更强调多个几何属性之间的内在一致性。相机参数、点图、深度图和三维轨迹并不是相互孤立的输出，而是共享同一场景几何的不同投影形式。将这些任务放到同一模型中联合预测，本质上是在利用多任务约束强化几何一致性\cite{wang2025vggt}。

这种统一输出思想具有两个直接优势。第一，它降低了传统流水线中接口错配的问题。过去不同模块之间往往因为特征表达、坐标系或误差累积方式不同而难以稳定耦合，而统一模型能够在内部自动协调这些关系。第二，它提高了结果的复用潜力。若同一模型已经预测出相机、点图和轨迹，那么这些中间结果便可以同时服务于重建、跟踪、深度估计乃至后续动态场景建模。

\subsection{与相关方法的横向比较}

为了更好理解 VGGT 的定位，可以将其与 DUSt3R 等近期方法进行对照。DUSt3R 也强调从图像中直接恢复三维几何，并在一定程度上简化传统结构恢复流程\cite{wang2024dust3r}。但与 VGG 团队这一主线相比，VGGT 更强调统一的几何属性输出，以及与前述几何先验、轨迹学习和可微 SfM 研究的连续性。

如果进一步与 MASt3R 比较，可以看到另一种重要差异。MASt3R 更强调把图像匹配问题锚定到三维空间中，从而改善匹配与几何恢复之间的联动\cite{mast3r2024}；而 VGGT 更进一步，希望把相机、深度、点图和轨迹一起纳入统一模型。这意味着前者更像是对“匹配与三维之间关系”的强化，后者则更像是对“整个几何系统输出接口”的统一。二者都体现出多视图几何正在从复杂流程走向统一表示，但关注点并不完全相同。

此外，\textit{Splatter Image: Ultra-Fast Single-View 3D Reconstruction}、\textit{RealFusion} 和 \textit{PC$^2$} 虽然聚焦单图条件下的三维恢复或生成先验，但它们同样体现出 VGG 团队对“前馈式三维表示”的持续探索\cite{szymanowicz2024splatter,melaskyriazi2023realfusion,melaskyriazi2023pc2}。这说明 VGG 团队并非只在传统多视图任务上推进学习化，也在尝试寻找更统一、更高效的三维表示方式。只是从本文主题出发，这些工作更适合作为研究边界上的支线，用于说明 VGG 团队在几何表示和三维生成之间建立联系的努力，而不宜替代多视图几何主线本身。

\subsection{向动态几何与统一表示扩展}

这一趋势还继续延伸到动态几何场景。\textit{Dynamic Point Maps: A Versatile Representation for Dynamic 3D Reconstruction} 说明，当场景随时间变化时，统一点图表示仍可作为多任务几何建模的重要中间层\cite{sucar2025dpm}。尽管这类工作已部分超出本文“多视图几何重建”的核心范围，但它们揭示了一个清晰方向：未来的学习方法可能不再局限于重建某个静态结果，而是形成可支持深度、相机、点云、轨迹乃至动态场景理解的统一几何表征。

这一扩展方向进一步说明，前馈式统一几何预测并不是研究终点，而更像一个平台性阶段。只要模型内部已经形成相对稳定的几何 token 或统一场景表示，那么从静态多视图重建扩展到视频、动态场景甚至跨任务三维推理，便成为顺理成章的下一步。这也是为什么动态点图等工作虽然在任务形式上已超出经典 SfM 范畴，但在方法论上仍与 VGG 的多视图几何学习主线保持一致。

\subsection{本阶段的综合评价}

从 VGG 团队近五年的方法演进来看，前馈式统一几何预测并不是凭空出现的，而是建立在前述几何先验注入、点轨迹学习与可微 SfM 逐步成熟的基础之上。也正因为如此，VGGT 等方法更适合被理解为一条研究主线的阶段性总结，而不是孤立的单篇突破。对本文主题而言，本阶段最值得强调的不是单个模型的性能，而是“多视图几何重建正在从优化驱动转向表示驱动与模型驱动”的方法论变化。

不过，这一阶段也暴露出新的挑战。其一，统一前馈模型虽然减少了显式优化，但训练成本与数据依赖往往更高。其二，模型输出越统一，越需要在训练阶段引入足够丰富且相互一致的监督信号，否则容易在某些属性上取得提升、在另一些属性上出现退化。其三，前馈式预测并不必然意味着完全替代几何优化，在高精度或复杂场景下，模型预测与几何求精之间仍可能形成新的混合范式。因此，从研究发展的角度看，未来很可能不是“优化彻底消失”，而是优化与统一表示之间出现新的分工结构。

从综述写作角度说，本阶段还提示我们不应把“前馈化”简单理解为“更快的工程实现”。真正值得强调的是，它改变了几何问题被组织、监督和复用的方式。过去多视图几何系统的中间变量主要服务于求解器本身，而现在统一模型输出的点图、轨迹和相机参数同时具备任务结果与通用表示的双重属性。这种变化使几何重建方法更容易向动态场景理解、具身感知和三维内容生成等领域外溢，也使 VGG 团队这一主线具有超出单一 benchmark 的方法论意义。
